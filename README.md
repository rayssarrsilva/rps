5. pull request it and wait for some feedback

## Description
A battle between YOU and the computer, both choose rock, paper or scissors each round and it is counted. 
At the end ONLY ONE of you, will win. GOOD LUCK!

## How to Play
1. open the html file
3. choose between rock, paper or scissors button and click
4. each round the score is updated
5. see if you won or lost at the end
